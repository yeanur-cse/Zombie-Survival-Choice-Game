# Zombie-Survival-Choice-Game
Description
A decision-based survival game where the player survives in a zombie world. Different user choices lead to different situations and outcomes.
Tools / Resources
Language: Java / C++ / Python
Libraries:
· Arrays
· Conditional Statements
· Random functions
IDE: Eclipse IDE or Visual Studio Code
Steps
· Start the game.
· Display a survival situation.
· Allow the user to choose an option.
· Process the user's choice.
· Generate the next event.
· Display the final result.
Architecture
Decision Game Architecture:
User Input · Decision Engine · Story Processing · Event Generator · Result Display
Outcome
A survival-based game that demonstrates how different decisions affect outcomes
